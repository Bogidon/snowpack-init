## Contributing

1. Clone the repo  
`git clone https://github.com/ndom91/create-pika-app`  

2. Install the project
`cd create-pika-app && npm install`

3. Link the binary
`npm link`

4. You should now be able to run `create-pika-app` from the command line to execute our `index.js`
